CoD(Call of Duty)

Call of Duty is one of the longest running FPS games on the market at this time. The first game of the series came out in 2003 and has maintained being one of the largest as well as oldest FPS in the market.

CoD Game Ratings
Solo Playability D-
Replayability B
Fun with Friends B-
Fun Solo F

EFT(Escape from Tarkov)

EFT is a Mil-Sim shooter game, it is more of a militaristic shooting game than the traditional shooters like CoD. This is a very extensive and intense game, this game will make you jump and scream with excitement.

EFT Game Ratings
Solo Playability C
Replayability C
Fun with Friends A+
Fun Solo C

Apex Legends

Apex Legends is a FPS as well as a BattleRoyale so it will fit into both categories. This game came out in 2019 and was off to a strong start with the prospect of a Pro League scene that was inevitably slowed drastically by COVID-19.

Apex Game Ratings
Solo Playability B+
Replayability B
Fun with Friends A+
Fun Solo C-
