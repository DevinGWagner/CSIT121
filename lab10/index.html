XML


What is XML?

XML stands for eXtensible Markup Language which uses markup language in order to store and transport data in a structured format able to be read by humans and machines.

How is XML used?

XML can be used in a number of ways such as for Data storage and retrieval by providing a standardized way to store and retrieve data, Configuring Files by allowing many software applications to use XML in order to store configuration data and for Web Services.

What does XML look like?

XML would look very similar to that of coding in HTML. It is able to be read by a person to understand what was done and the computer is able to turn the information into a table or another form of website layout.

What is JavaScript?

JavaScript is more of a high-level, another typed programming language which is primarily used for creating interactive client-side web applications. JavaScript is very verastile in the fact that it can be used for web developement, server-side programming, game developement and mobile application developement.

How is JavaScript related to HTML and CSS?

JavaScript, HTML, and CSS are three core technologies used in web development, and they work together to create dynamic and interactive web pages.

What are some common JavaScript elements, properties, methods, and functions?

An Element of JavaScript would be loops which are used to continuously execute code repeatedly. A property of JavaScript would be style which is similar to HTML by getting or setting the CSS style of an element. One method would be console.log(): which would log a message to the browsers console for debugging purposes. Function would be confirm(): displays a message in a dialog box and prompts the user for confirmation.
